## Opdracht 1 - Netwerken en Security vervolg

Om dit programma te bouwen/runnen, moet docker geinstalleerd zijn.

Om te beginnen moet je in de folder zitten waar het `docker-compose.yml` bestand ook in zit

Vervolgens moet in deze map een terminal geopend worden.

<b>Stap 1</b> Run het volgende commando
- Linux/Mac: `sudo docker compose up -d --build`
- Windows: `docker compose up -d --build`
